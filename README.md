# Website Quick Reference
## Author: Nguyen Trung Nhan @nhannt201
Source code of extension Website Quick Reference
#### This utility helps you to extract website sources according to Harvard standards quickly!
#### Quick reference is a tool to help you quickly cite website references according to Harvard citation standards. Usage is very simple, just click on the icon of the tool at the website you want to cite the source.
#### Link: https://chrome.google.com/webstore/detail/website-quick-reference/oeilagkbjpeojpkidlbdfhpecchcohmf
